# Python 120 Coding Questions

A structured collection of **120 Python coding questions** designed to help beginners build a strong foundation in Python programming through hands-on practice.

The questions are organized into **4 weekly notebooks**, covering Python fundamentals from basic syntax to Object-Oriented Programming (OOP).

## 📚 Course Structure

### Week 1: Python Basics

* Variables and Data Types
* Input and Output
* Operators
* Conditional Statements
* Basic Programming Logic

📄 `week1/basics.ipynb`

---

### Week 2: Functions and Data Structures

* Functions
* Lists
* Tuples
* Dictionaries
* Common Operations and Practice Questions

📄 `week2/Functions,Lists,Tuples,Dictionary.ipynb`

---

### Week 3: File Handling and Error Handling

* Reading and Writing Files
* File Operations
* Exception Handling
* Try, Except, Finally
* Practical Exercises

📄 `week3/File_and_error_handling.ipynb`

---

### Week 4: Object-Oriented Programming (OOP)

* Classes and Objects
* Constructors
* Inheritance
* Polymorphism
* Encapsulation
* OOP Practice Problems

📄 `week4/oops.ipynb`

---

## 🎯 Objective

This repository is intended for:

* Python beginners
* Students preparing for coding interviews
* Anyone looking to strengthen Python fundamentals through practice

By completing all 120 questions, learners will gain practical experience with core Python concepts and develop problem-solving skills.

## 🚀 How to Use

1. Clone the repository:

   ```bash
   git clone https://github.com/Adarsh-0417/Python_120_Quest.git
   ```

2. Open the notebooks using Jupyter Notebook or VS Code.

3. Follow the weekly progression and solve the questions sequentially.

## 🛠 Prerequisites

* Python 3.x
* Jupyter Notebook / Jupyter Lab
* VS Code (optional)
